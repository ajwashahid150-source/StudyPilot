
# 📚 StudyPilot

StudyPilot is a console-based student productivity application developed in Python. It helps students manage their accounts and organize study tasks through a simple menu-driven interface. The project demonstrates file handling, authentication, task management, and modular programming concepts.

---

## 🚀 Features

- 👤 Student Registration
- 🔐 Secure Login System
- 🔑 Change Password
- 👨‍🎓 View Student Profile
- 📖 Study Planner
- ➕ Add Study Tasks
- 📋 View All Tasks
- ✅ Mark Tasks as Completed
- 🗑 Delete Tasks
- 🚪 Logout System
- 📂 Persistent Data Storage using Text Files

---

## 🛠 Technologies Used

- Python
- File Handling
- Conditional Statements
- Loops
- Lists
- Functions
- Menu-Driven Programming
